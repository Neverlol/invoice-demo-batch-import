# 智能开票助手现场操作说明

> 这份说明给现场助理使用。文件名全部使用英文/数字，是为了避免 Windows 解压中文文件名乱码。

---

## 一、第一次在这台电脑使用

如果需要连接 MiMo、阿里云同步、云端客户档案，请先安装现场私密配置：

1. 把私密配置包解压到当前文件夹。
2. 确认存在：`_onsite_private_config/onsite_secrets.json`。
3. 双击：`01_INSTALL_PRIVATE_CONFIG.bat`。
4. 安装成功后，建议删除 `_onsite_private_config` 文件夹。

然后双击：

```text
00_FIRST_INSTALL.bat
```

它会自动检查并准备：

- Python 运行环境
- `.venv` 虚拟环境
- 开票助手依赖包
- Edge 浏览器连接能力
- OCR 组件
- 桌面快捷方式

安装完成后，桌面会出现：

```text
智能开票助手
```

## 二、每天怎么启动

推荐方式：

```text
双击桌面：智能开票助手
```

如果没有桌面快捷方式，就双击当前文件夹里的：

```text
02_START_INVOICE_ASSISTANT.bat
```

如果一键启动不稳定，也可以分两步：

```text
03_START_TAX_BROWSER.bat
04_START_WORKBENCH.bat
```

工作台地址：

```text
http://127.0.0.1:5012
```

注意：不要关闭弹出的黑色窗口，也不要关闭税局专用浏览器。

## 三、登录税局

在工作台首页点击：

- 打开辽宁税局
- 打开吉林税局
- 打开北京税局

根据当前客户地区选择。

登录成功后，回到工作台点击：

```text
识别当前税局主体 / 加载档案
```

看到当前主体名称后，再继续生成草稿。

## 四、单张发票怎么开

1. 确认“销售方 / 纳税主体”。
2. 粘贴客户开票信息，或上传图片 / Excel / PDF。
3. 点击“生成开票草稿”。
4. 进入草稿页后检查：
   - 购买方名称
   - 购买方税号
   - 项目名称
   - 含税金额
   - 税率
   - 税收编码
5. 如需修改，直接改字段并保存。
6. 确认无误后，发起开票 / 上传税局。
7. 税局到“预览发票”后，人工确认。

禁止：未经客户确认，不要点击最终开具。

## 五、多张截图怎么批量开

适用于餐饮平台截图、多个购买方、多张发票集中开具。

1. 首页填写或确认销售方 / 纳税主体。
2. 勾选“批量开具发票”。
3. 一次性上传多张截图。
4. 点击“生成开票草稿”。
5. 系统进入“批量开票工作表”。

批量页会看到：

```text
Sheet 1：发票基本信息
Sheet 2：发票明细信息
Sheet 3：待补全 / 异常项
```

重点检查：

- 一张截图是否对应一张发票。
- 购买方名称是否完整。
- 购买方税号是否正确。
- 含税金额是否正确。
- 项目、税率、税收编码是否合理。

字段旁边有 `✎` 表示可以直接编辑。

修改后点击：

```text
保存本批修改 / 重新校验
```

然后可以：

```text
下载本批税局 Excel
发起本批开票 / 上传税局
```

## 六、现场安全边界

必须遵守：

- 系统只到税局“预览发票”。
- 不自动最终开具。
- 最终开具必须由客户 / 助理人工确认。
- 不在群里发送税局账号、密码、验证码、登录二维码。
- 截图如果包含敏感信息，请先打码。

## 七、常见问题

### 工作台网页打不开

先确认已经双击：

```text
04_START_WORKBENCH.bat
```

然后手动打开：

```text
http://127.0.0.1:5012
```

### 税局无法连接

确认已经双击：

```text
03_START_TAX_BROWSER.bat
```

并且税局浏览器没有关闭。

### 批量截图数量不对

确认首页已经勾选：

```text
批量开具发票
```

然后重新上传图片并生成草稿。
