# 智能开票助手交付包说明

## 一、给现场助理看的文件

第一次在新电脑使用，先双击：

- `首次安装智能开票助手.bat`

安装完成后，日常使用优先双击桌面快捷方式：

- `智能开票助手`

如果没有桌面快捷方式，也可以双击文件夹里的：

- `启动智能开票助手.bat`

如果需要分步启动：

- `①启动税局浏览器.bat`
- `②启动开票工作台.bat`
- `③查看现场操作说明.bat`

## 二、首次安装会做什么

`首次安装智能开票助手.bat` 会自动检查并准备：

- Python 3.11
- Python 虚拟环境 `.venv`
- 开票助手依赖包
- Microsoft Edge 检测
- OCR 组件 Tesseract（如果包内安装器可用）
- 桌面快捷方式 `智能开票助手`

安装依赖时会优先使用包内离线 wheels；如果离线安装失败，才会尝试联网安装。

## 三、不建议现场助理操作的文件

以下文件是技术人员维护使用，现场助理不用打开：

- `app.py`
- `start_lean_workbench.py`
- `tax_invoice_demo/`
- `tax_invoice_batch_demo/`
- `templates/`
- `static/`
- `tools/`
- `.venv/`
- `output/`
- `windows_bootstrap/`

## 四、现场交付建议

建议把整个文件夹放到：

`C:\智能开票助手\`

然后让助理第一次双击：

`首次安装智能开票助手.bat`

安装完成后，桌面会生成快捷方式：

`智能开票助手`

以后每天只需要双击桌面这个快捷方式。
