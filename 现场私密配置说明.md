# 现场私密配置说明

## 为什么要分成两个包

智能开票助手现场交付分为：

1. 公开安装包：代码、依赖、启动脚本、说明文档，不含 token。
2. 私密配置包：MiMo Key、阿里云 sync token、tenant，只给可信安装人员或可信助理。

这样可以避免把 token 打进 Git、公开 zip、微信群或客户资料包。

## 私密配置包目录结构

把私密配置包解压到智能开票助手根目录后，应看到：

```text
_现场私密配置/
  onsite_secrets.json
```

然后双击：

```text
安装现场私密配置.bat
```

安装器会生成：

```text
llm_client.local.json
sync_client.local.json
```

这两个文件已经在 `.gitignore` 中，不应提交 Git。

## onsite_secrets.json 格式

请复制 `onsite_secrets.example.json`，改名为：

```text
_现场私密配置/onsite_secrets.json
```

然后只在本机私密环境里填入真实值。

必填字段：

```json
{
  "mimo_api_key": "真实 MiMo Key",
  "sync_endpoint": "http://39.106.111.126:5021/api/invoice/events",
  "sync_token": "真实 sync token",
  "sync_tenant": "liaoning-seed-20260506"
}
```

建议设置：

```json
{
  "delete_source_after_install": true
}
```

这样安装成功后会自动删除 `_现场私密配置/onsite_secrets.json`。

## 安全建议

- 不要把 `onsite_secrets.json` 发微信群。
- 不要提交 Git。
- 不要放进公开安装包。
- 私密配置包建议单独加密压缩并通过可信渠道传递。
- 现场 token 应使用专用 token，权限只覆盖当前 tenant。
- 现场测试结束后，可以在阿里云侧吊销或轮换 token。

## 安装顺序

新电脑推荐顺序：

```text
1. 解压公开完整安装包到 C:\智能开票助手\
2. 解压私密配置包到同一目录
3. 双击 安装现场私密配置.bat
4. 双击 首次安装智能开票助手.bat
5. 日常使用双击桌面 智能开票助手
```
